Opzione 1:
I numeri da 1 a 6 antecedenti ai file SQL specificano l'ordine per effettuare un corretto caricamento consecutivo dei relativi file, 
al fine di procedere con una corretta installazione del DB in un ambiente adatto a tale scopo.

Opzione 2:
Il file "database.sql" è una raccolta degli altri file SQL, dunque per l'importazione e l'installazione corretta del DB 
è sufficiente il caricamento di quest'utlimo in un ambiente adatto a tale scopo.
